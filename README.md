# RAG PoC

Local RAG (Retrieval-Augmented Generation) proof of concept for learning purposes.
Simulates querying SOC documentation (Confluence-style) using a local LLM.

## Stack
- **Embedding**: sentence-transformers (local)
- **Vector store**: ChromaDB
- **LLM**: Ollama + Mistral 7B
- **Interface**: FastAPI + HTML

## Setup
Requires VS Code with Dev Containers extension. Open this folder and select "Reopen in Container".
